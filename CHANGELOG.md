# CHANGELOG

## [v2.1.0] — 2026-06-27

### ✨ 新功能

- **自然语言触发**：对称支持 `对称左/右/上/下/左上/右上` 及 `对称 左` 等文字参数，无需记数字
- **自然语言触发**：真对称支持 `真对称右/左/上/下` 及 `右真对称/左真对称/上真对称/下真对称`
- **独立指令**：新增 `左上对称`、`右上对称` 两个独立指令
- **波普联画动图**：多联画模式（panels>1）现在也支持 GIF 输入，每帧走完整联画管道
- **紧凑语法全覆盖**：所有单参命令（旋转/故障/万花筒/抖动/呼吸/包浆/电子包浆/马赛克/裸眼3d/加速/调速）全部支持无空格触发

### 🔧 调整

| 项目 | 旧默认 | 新默认 |
| :--- | :--- | :--- |
| `match_mode`（无需`/`模式） | `false` | `true` |
| `gif_speed_allow_frame_drop`（允许丢帧） | `false` | `true` |
| `default_speedup_factor`（加速倍率） | `2.0` | `2.5` |
| **对称默认** | type=2（上对称） | type=1（左对称） |
| 波普 `num_colors` / `edge_width` / `panels` | `6` / `3` / `4` | `8` / `1` / `2` |
| 裸眼3D `line_spacing` / `line_width` / `mask_threshold` | `80` / `3` / `25` | `40` / `4` / `15` |

### ♻️ 重构

- **图片来源优先级**：引用图 > 内联图 > @头像（之前 `@` 优先于引用）
- **全部数值参数加兜底范围**：防止极端值导致插件崩溃
- **重命名**：`哈哈镜` → `扭曲`，避免关键词冲突
- **仓库重命名**：`astrbot_plugin_pic_toolbox` → `astrbot_plugin_kimage`

### 🐛 修复

- `波普` 支持紧凑语法 `波普8`
- `对称` 新增类型 5=右对称、6=下对称，补全所有方向

### 📝 文档

- 全面更新 README 命令表、配置表格、项目结构
- 更新 CHANGELOG

## [v2.0.0] — 2026-06-27

### ✨ 新增指令（10 个）

| 指令 | 说明 |
| :--- | :--- |
| `旋转 [角度步长]` | 圆形旋转 GIF（裁内切圆，圆外透明） |
| `故障 [强度]` | RGB 通道偏移，赛博故障风 GIF |
| `万花筒 [扇区]` | 扇区镜像对称，旋转迷幻 GIF |
| `抖动 [颜色数]` | 降色阶 + Floyd-Steinberg 抖动，复古 8-bit |
| `呼吸 [帧延迟]` | 周期缩放呼吸动画 |
| `包浆 [强度]` | Sepia 暖黄 + 暗角 + 噪点 + 褪色，老照片做旧 |
| `波普 [格数]` | Pop Art，降色 + Canny 描边 + 半调网点 + 多联画（默认 4 格） |
| `电子包浆 [程度]` | JPEG 伪影 + 色彩断层 + 像素化，互联网蹂躏感 |
| `扭曲 [类型] [强度]` | 6 种曲面扭曲（凸/凹/横柱/竖柱/波浪/螺旋），支持 1-6 数字 |
| `对称[1-4]` | 统一对称入口（1=左 2=上 3=\\ 4=/），支持紧凑语法 `对称1` |

### ✨ 已有指令增强

| 指令 | 增强内容 |
| :--- | :--- |
| `对称[1-4]` | 新增 `对称3` \\ 对角线对称、`对称4` / 对角线对称 |
| `真对称[1-4]` | 新增延展型对称，画布扩大（宽或高翻倍），支持 GIF |
| `裸眼3d [强度]` | 新增强度参数 1-10，自动映射 line_spacing/width/threshold |
| `帮助` | 新增别名 `图帮助`、`图help`、`img_help` |
| `往返` | 从 imgtool 移植：GIF 正序+倒序 ping-pong 循环 |
| `裸眼3d` | 从 imgtool 移植：分层假象裸眼3D（8 参数可配置） |

### 🔧 重要变更

- **紧凑参数语法**：`对称1` `扭曲1 0.5` `旋转12` 等支持不带空格触发
- **带参触发**：旋转/故障/万花筒/抖动/呼吸/包浆/波普/电子包浆/扭曲/裸眼3d 共 10 指令支持内联参数
- **扭曲类型映射**：1=凸 2=凹 3=横柱 4=竖柱 5=波浪 6=螺旋
- **对称默认值**：`对称` 无参数默认 type=1（左对称）

### 🐛 修复

- **GIF 全支持**：修复 spin.py / glitch.py / kaleidoscope.py / breathing.py 在 GIF 输入时只取首帧的问题，现全部支持逐帧处理，输出组合动画
- **波普默认**：从 1 联改为 4 联画

### 📝 文档

- README 全面重写，新增所有指令说明、配置项表格、项目结构
- CHANGELOG 本文件更新
